## Lab Template

Every lab follows this structure:

- `README.md` — Objective, prerequisites, architecture, walkthrough, validation, cleanup
- `commands.md` — The exact `gcloud` / shell commands executed
- `screenshots/` — Console screenshots that demonstrate the result
- One lab-specific notes file (e.g. `lessons-learned.md`, `security-notes.md`)
