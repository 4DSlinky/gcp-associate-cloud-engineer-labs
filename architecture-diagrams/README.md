## Tools I recommend

- [draw.io / diagrams.net](https://www.diagrams.net/) — free, has GCP icon set
- [Excalidraw](https://excalidraw.com/) — quick, sketchy diagrams
- [Google Cloud architecture icons](https://cloud.google.com/icons) — official icon library

## Conventions

- Show **trust boundaries** (VPC, project, internet) explicitly
- Label each arrow with the protocol and port
- Mark public-facing components in a distinct color
